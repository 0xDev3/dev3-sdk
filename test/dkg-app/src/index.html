Storing a DKG Asset using Dev3 SDK

This example shows how to store an asset on the DKG using nothing but the Dev3 platform.

Prerequisits:

  1. Workspace created on the app.dev3.sh on the Gnosis Chain. API key and Project ID must be configured in the `index.mjs` file in the project root.
  2. Having an account funded with some native XDAI for executing transactions
  3. Having some TRAC token for storing a knowledge asset

Hit connect wallet button below. This will connect wallet to the app and check if Dev3 API is successfully initialized. It also checks for TRAC balance on the connected wallet.
